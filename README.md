# Calculator CLI v1.0 – Python

A beginner-friendly command-line calculator built in Python.  
Supports basic arithmetic, advanced math, and percentage tools, with modular code design.

## Features
- Basic operators (+, −, ×, ÷)
- Advanced math: square, square root, exponent
- Percentage calculations: %, increase %, decrease %
- Input validation & rounding
- Modular function design (`math_functions.py`)

## Future Improvements (Planned)

* More functions: factorial, logarithm, trigonometry
* History of calculations in `.txt` file
